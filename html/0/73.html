经验总结:建一个网站，需要懂得很多(不断更新中...)

1. 建站技术

1.1 购买域名
godday、万网、美橙、空间商提供的 需要50元/年
1.2 购买空间
blogbus、godday、vps、ramhost、sina app、阿里云等 100元/年 -> 50元/月 -> ？
1.3 备案
国外空间商无需备案、国内空间商需要，这个就是麻烦、太麻烦

2. 建站程序

2.1 建站语言
需要懂得 css、javascript、php/asp/jsp、html布局、数据库SQL语言
2.2 建站程序
wordpress、zblog、dizcuss、wiki等根据自己的需要有免费的、有收费的
2.3 辅助程序
dreamweam、editplus、notepad++、js调试器、css调试、浏览器、程序加载检测器、gimp、photoshop、素材设计
2.4 第三方应用
统计：51啦、百度统计、站长统计。
分享：百度分享、JiaThis、bshare。
评论：多说、评论啦、友言
插件：天气插件、微博插件、无觅相关文章推荐

3. 网站优化

3.1 页面美工
专业啊，这个需要有细胞的，主要就是图片、css、flash等
3.2 访问速度
精简css和javascript、减少数据库访问、减少图片的数量大小、选择好的网速、好的主机等

4. 推广SEO

4.1 搜索引擎
百度竞价排名、google推广、提高搜索引擎收录量、提高搜索排名、提高PR值、提高alex排名、交换友情链接、做链接联盟。
4.2 分享
方法：无觅分享、Jiathis、百度分享
4.3 统计分析
google分析、百度分析、51.la、淘宝分析
4.4 抓住读者
写文章、做线下活动、推广文章、合适读者口味文章;更新周期合适；主题合适
4.5 博客圈子
关注论坛：28推、wordpres.org.cn、站长论坛
关注名博：卢松松、willian
关注同类博客：google订阅、抓虾订阅，九点订阅、freebrun订阅。
互动：博客留言、论坛留言、参加活动

5. 网站盈利

广告联盟(淘宝阿里妈妈、google adsence、拿福能、博客联盟、百度联盟)
返佣（cps返佣、淘宝客返佣）
用户主动广告
搜索收入（百度、google）
阿里云推广
pr值赚钱等
不断
