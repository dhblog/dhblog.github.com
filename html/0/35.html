1 起源

首先感谢自说me话：http://isayme.com/2011/06/handsome-in-my-blog/ 这篇文章讲了它弄了个帅哥认证，虽然我没有被评为帅哥，但是这不妨碍我学习并定义我这里的帅哥。

2 方法

方法很简单,在functions.php中添加如下语句 
function comment_g_title($email = '')
{
	if(empty($email))return;
	$handsome=array(
	'1'=>'aa@bb.com', 
	'2'=>'cc@dd.com');

	$adminEmail = get_option('admin_email');
	if($email==$adminEmail)
	echo " 
博主
";
	elseif(in_array($email,$handsome))
	echo " 
帅哥
";
}
aa@bb.com 为需要添加帅哥的人的邮件。另外需要在评论的显示评论者名字的地方加上一句 
comment_author_email); ?>

3 帅哥评选原则

在我这里评论不小于10条的为帅哥，iSayMe虽然只有9个，但是用了他的东西，所以给他例外。 这里给出评选的SQL语句： 
SELECT comment_author, comment_author_email, COUNT( * ) 
FROM `wp_comments` 
where comment_approved = '1'
GROUP BY comment_author
ORDER BY COUNT( * ) DESC 
LIMIT 0 , 30
SQL语句执行结果图： 很激动 松哥在我这里的留言超过10条了。
